数组的内存地址是连续的

链表的元素可以在内存的任何地方, 每个元素存了下一个元素的地址

关于操作数组和链表的时间复杂度:

| --   | 查询   | 插入   | 删除   |
| ---- | ---- | ---- | ---- |
| 数组   | O(1) | O(n) | O(n) |
| 链表   | O(n) | O(1) | O(1) |

从上可以看出, 数组在查询上可以更快, 链表在插入和删除上更快, 为了结合数组和链表的优点, 有同时使用的情况, 比如一个网站的用户注册, 可以以`A-Z`为数组, 在每个字母后面加入链表, 这样可以在添加新用户的时候能快速找到要添加的链表并进行插入, 同时在查询用户的时候也能缩短查询时间

#### 单向链表

单向链表需要满足一下方法

- 链表大小size
- 链表是否为空
- 插入元素
  - 头部插入元素
  - 尾部插入元素
- 删除元素
- 返回一个指定元素的位置

```python
#!/usr/bin/env python
# coding:utf-8

class Node(object):
    """定义节点"""
    __slots__ = ['_item', '_next']

    def __init__(self, item):
        self._item = item
        self._next = None

    def getItem(self):
        return self._item

    def setItem(self, newitem):
        self._item = newitem

    def getNext(self):
        return self._next

    def setNext(self, newnext):
        self._next = newnext


class SingleLinkedList(object):
    def __init__(self):
        self._head = None
        self._size = 0

    def isEmpty(self):
        """检测链表是否为空"""
        return self._head == None

    def size(self):
        """链表大小"""
        count = 0
        current = self._head
        while current != None:
            count += 1
            current = current.getNext()
        return count

    def add(self, item):
        """在头部添加元素"""
        temp = Node(item)
        temp.setNext(self._head)
        self._head = temp

    def append(self, item):
        """在尾部添加元素"""
        temp = Node(item)
        if self.isEmpty():  # 如果是空表, 将添加的元素设为第一个元素
            self._head = temp
        else:
            current = self._head
            # 直到下一个节点为空
            while current.getNext() != None:
                current = current.getNext()
            current.setNext(temp)

    def search(self, item):
        """在链表中查找元素"""
        current = self._head
        status = False
        while current != None and not status:
            if current.getItem() == item:
                status = True
            else:
                current = current.getNext()
        return status

    def index(self, item):
        """找出元素在链表的位置"""
        current = self._head
        count = 0
        found = False
        while current != None and not found:
            count += 1
            if current.getItem() == item:
                found = True
            else:
                current = current.getNext()
        if found:
            return count
        else:
            return '没有发现', item

    def remove(self, item):
        """
        删除元素
        要记录前一个节点, 因为next要指向后一个节点
        """
        current = self._head
        pre = None
        while current != None:
            if current.getItem() == item:
                # 判断更改的元素是不是第一个
                if not pre:
                    self._head = current.getNext()
                else:
                    pre.setNext(current.getNext())
                break
            else:
                pre = current
                current = current.getNext()

    def insert(self, pos, item):
        """插入元素"""
        if pos <= 1:
            self.add(item)
        elif pos > self.size():
            self.append(item)
        else:
            temp = Node(item)
            count = 1
            pre = None  # 记录前一个指针的位置
            current = self._head
            while count < pos:
                count += 1
                pre = current
                current = current.getNext()
            pre.setNext(temp)
            temp.setNext(current)

    
if __name__ == "__main__":
    s = SingleLinkedList()
    s.append(10)
    s.append(20)
    s.append(30)
    print s.size()
    print s.index(20)
```
