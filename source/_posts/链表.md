数组的内存地址是连续的

链表的元素可以在内存的任何地方, 每个元素存了下一个元素的地址


数组在查询上可以更快, 链表在插入和删除上更快, 为了结合数组和链表的优点, 有同时使用的情况, 比如一个网站的用户注册, 可以以`A-Z`为数组, 在每个字母后面加入链表, 这样可以在添加新用户的时候能快速找到要添加的链表并进行插入, 同时在查询用户的时候也能缩短查询时间

#### 单向链表

单向链表需要满足一下方法

- 链表大小size
- 链表是否为空
- 插入元素
  - 头部插入元素
  - 尾部插入元素
- 删除元素
- 返回一个指定元素的位置

```python
# coding:utf-8
# __author__ = 'qshine'


class Node(object):
    """定义节点"""

    def __init__(self, value, next_=None):
        self.value = value
        self.next = next_


class SingleLinkedList(object):
    """定义链表方法"""

    def __init__(self):
        self._head = None

    def is_empty(self):
        """判断是否为空链表"""
        return self._head == None

    def size(self):
        """链表大小"""
        current, size = self._head, 0
        while current != None:
            size += 1
            current = current.next
        return size

    def prepend(self, value):
        """头部加入元素"""
        temp = Node(value)
        temp.next = self._head
        self._head = temp

    def append(self, value):
        """尾部加入元素"""
        temp = Node(value)
        if self._head == None:
            self._head = temp
        else:
            current = self._head
            while current.next != None:
                current = current.next
            current.next = temp

    def insert(self, position, value):
        """插入元素"""
        if position <= 1:
            self.prepend(value)
        elif position >= self.size():
            self.append(value)
        else:
            temp = Node(value)
            pre, current, num = None, self._head, 1
            while num <= position:
                pre = current
                current = current.next
                num += 1
            pre.next = temp
            temp.next = current


    def delete(self, value):
        """删除元素"""
        current = self._head
        pre = None
        while current != None:
            if current.value == value:
                # 判断要删除的元素是不是第一个
                if not pre:
                    self._head = current.next
                else:
                    pre.next = current.next
                break    # 删除后及时跳出
            else:
                pre = current
                current = current.next

    def pop_first(self):
        """头部弹出元素"""
        if self._head == None:
            raise ValueError('链表为空')
        res = self._head.value
        self._head = self._head.next
        return res

    def pop_last(self):
        """从尾部弹出元素"""
        if self._head == None:
            raise ValueError('链表为空')
        current = self._head
        # 如果只有一个元素
        if current.next == None:
            value = current.value
            self._head = None
            return value
        while current.next.next != None:
            current = current.next
        value = current.next.value
        current.next = None
        return value


    def find(self, value):
        """查找元素是否在链表中"""
        status = False
        current = self._head
        while current != None and not status:
            if current.value == value:
                status = True
            else:
                current = current.next
        return status

    def printAll(self):
        current = self._head
        while current != None:
            print(current.value)
            current = current.next





if __name__ == '__main__':
    s = SingleLinkedList()
    s.append(10)
    s.append(20)
    s.append(30)
    s.prepend(5)
    print(s.size())
    print('-----')
    s.printAll()
    s.delete(20)
    print('-----')
    s.printAll()
    s.insert(2, 111)
    print('-----')
    s.printAll()


```
