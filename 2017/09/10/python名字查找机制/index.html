python名字查找机制

Sep 10, 2017 | Python

python中对任何变量的创建, 查找, 修改都是在命名空间(namespace)进行的, 命名空间分为

  • 局部作用域(Local)
    如函数内部的变量声明
  • 全局作用域(Global)
    定义在python模块中的变量名

  • 嵌套作用域(enclosing function locals)
    多重函数嵌套情况, 嵌套情况下global并不起作用

  • 内置作用域(Build-in)

python中访问一个变量的时候遵循LEGB原则(本地 –> 嵌套 –> 全局 –> 内置), 在第一个找到的地方停止搜索
