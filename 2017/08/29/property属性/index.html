Aug 29, 2017 | Python

property是实现数据可管理性的内建数据类型, 其实质是一种特殊的数据描述符(实现了__get__()和__set__方法, 称为数据描述符)

property实现形式为property(fget=None, fset=None, fdel=None, doc=None)
