python模块-itertools

Aug 11, 2017 | Python

itertools模块所包含的所有函数都将构造和返回迭代器, 迭代器相比列表来说好处是延迟计算, 所以此模块返回的迭代器都可以与for语句结合使用

注意: python2和python3中某些函数有变化, 具体参考官方文档

chain(iter1, iter2, …, iterN)

将多个迭代器联合起来并返回一个新的迭代器

accumulate(iter), python3存在

累加

combinations(itertable, n)

返回itertable中所有长度为n的子序列, 顺序为itertable的顺序, 不允许重复

combinations_with_replacement(iterable, n)

返回iterable中长度为n的子序列, 允许元素重复

islice(iterable, start, stop, [step])

用于截取迭代器

count(start=0, step=1)

一个计数器, 可以指定起始位置和步长, 没有结束位置, 所以和islice`结合

cycle(itertable)

循环指定的itertable

dropwhile(predicate, iterable)

从iterable中删除元素, 只要元素符合predicate的条件

groupby(iterable[, key])

对iterable按照指定条件进行分组

ifilter(predicate, iterable)

从iterable中筛选出条件为True的元素

ifilterFalse(predicate, iterable)

筛选出条件为Flase的元素, 注意python3中为filterflase

permutations(iterable[,n])

从iterable中返回指定长度n的所有组合

product(iterable[,repeat])

多个可迭代对象的笛卡尔积

该模块还有很多其它方法, 具体参考官方文档
