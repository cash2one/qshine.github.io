python类的构造方法__new__

Aug 24, 2017 | Python
  • __new__: 类的构造方法, 当返回类的对象的时候会自动调用__init__进行初始化, 没有对象返回的话则不会调用
  • __init__: 类的对象创建好后进行变量初始化的方法

上例可以看出首先运行的是__new__方法, 创建好类的对象后才运行__init__方法初始化变量

总结

  • __init__()并不是类的构造方法, __init__()的工作是在类的对象创建好后进行变量的初始化
  • __new__()方法是真正的创建实例, 是类的构造方法
  • 当需要覆盖__new__()和__init__()方法的时候要保证参数一致
  • 当子类继承自不可变类型(str, tuple)等类型或使用工厂方法的时候需要覆盖该方法
