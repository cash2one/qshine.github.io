三. 发布项目

接着在另一个终端使用打包项目, 还是进入项目根目录:scrapyd-deploy <deploy-name> -p <project-name>

此时就已经打包成功, 再次查看目录

四. 调度爬虫

命令curl http://localhost:6800/schedule.json -d project=default -d spider=somespider

  • default是在scrapy.cfg中project的名字
  • somespider是spider的名字

执行后就可以在浏览器中打开127.0.0.0:6800查看执行情况, 日志等信息.
