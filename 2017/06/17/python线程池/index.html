python线程池

Jun 17, 2017 | Python | 阅读

线程的生命周期: 创建, 就绪, 运行, 阻塞, 终止. 一个线程的运行时间分为三部分:

  • 启动时间
  • 运行时间
  • 销毁时间

线程池适合处理突发性大量请求或者需要大量线程来完成的任务, 但是任务时间比较短的场景, 能有效避免创建过多线程导致系统负荷过大, 响应过慢的情况, 同时也能避免线程不断创建和销毁浪费的时间

如果仅仅需要创建少数线程, 可以直接全部创建, 但是任务量很大的时候线程池很有效
